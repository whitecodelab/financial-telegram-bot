Financial Telegram Bot

Telegram бот для учета личных финансов с расширенной статистикой и визуализацией.

ВОЗМОЖНОСТИ:
- Учет доходов и расходов
- Визуальная статистика (диаграммы и графики)
- Персональные категории расходов
- Редактирование операций
- Интуитивный интерфейс с кнопками
- Локальное хранение данных в SQLite

БЫСТРЫЙ СТАРТ:

1. Установите зависимости:
pip install -r requirements.txt

2. Настройте бота:
- Создайте бота через @BotFather
- Получите API токен
- Создайте файл config.py с содержимым:
API_TOKEN = "your_bot_token_here"

3. Запустите бота:
python telegram_bot.py

СТРУКТУРА ПРОЕКТА:
telegram_bot.py - основной код бота
sqlite_database.py - работа с базой данных
categories.py - система категорий
charts.py - визуализация данных
config.py - конфигурация
requirements.txt - зависимости

ИСПОЛЬЗОВАНИЕ:
