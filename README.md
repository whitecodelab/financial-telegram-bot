Financial Telegram Bot

Telegram бот для учета личных финансов с расширенной статистикой и визуализацией.

ВОЗМОЖНОСТИ:
- Учет доходов и расходов
- Визуальная статистика (диаграммы и графики)
- Персональные категории расходов
- Редактирование операций (сумма, описание, категория)
- Интуитивный интерфейс с inline-кнопками
- Локальное хранение данных в SQLite

БЫСТРЫЙ СТАРТ:

1. Установите зависимости:
pip install -r requirements.txt

2. Создайте config.py с токеном бота:
API_TOKEN = "your_bot_token_here"

3. Запустите бота:
python telegram_bot.py

СТРУКТУРА ПРОЕКТА:
telegram_bot.py - основной код бота
sqlite_database.py - работа с базой данных
categories.py - система категорий
charts.py - визуализация данных
config.py - конфигурация
requirements.txt - зависимости

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ:
Добавить расход: 500 продукты
Добавить доход: +50000 зарплата

КОМАНДЫ:
/start - главное меню
/stats - статистика
/chart - диаграмма расходов
/balance - баланс
/list - список операций

ТЕХНОЛОГИИ:
Python 3.7+, pyTelegramBotAPI, SQLite3, Matplotlib
